# Robotics & Spatial Intelligence Brief

A readable digest of what's happening in robotics worldwide and in spatial intelligence: world models, 3D perception, embodied AI, and the labs driving it.

**Live site:** [herronoui.github.io/robotics-spatial-brief](https://herronoui.github.io/robotics-spatial-brief)

## What's inside

- **Spatial intelligence explainer** — plain-language overview of 3D perception, world models, spatial reasoning, and embodied AI
- **Worldwide snapshot** — regional highlights (US, Europe, Asia, Japan/Korea)
- **Trend cards** — humanoids, world models, VLAs, physical AI funding
- **Live feeds** — auto-categorized news from arXiv, MIT, IEEE, industry media

## Update the digest

```bash
python3 scripts/fetch_digest.py
git add data/digest.json && git commit -m "Update digest" && git push
```

## Run locally

```bash
open index.html
```
